# util_TestHarness
Module to create fake RFQ's, iterating over all possible values
of properties of an RFQ.

## Installation
```bash
git clone git@spokedev.githost.io:spokedev/util_rfqharness.git
cd util_testHarness
npm i
```

## Usage
```bash
npm start
Spoke|- generate Rfqs
```

## Test
```bash
npm test
```
